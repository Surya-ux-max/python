# VijayGPT - Terminal Edition\n\nAn interactive terminal-based question-answering system about Thalapathy Vijay using AI-powered semantic search.\n\n## Features\n\n- Interactive terminal interface\n- AI-powered semantic search using sentence transformers\n- Knowledge base about Thalapathy Vijay's career, movies, and politics\n- Confidence scoring for answers\n- Easy-to-use command interface\n\n## Requirements\n\n- Python 3.7+\n- sentence-transformers\n- scikit-learn\n- numpy\n\n## Installation\n\n1. **Install dependencies**\n   ```bash\n   pip install sentence-transformers scikit-learn numpy\n   ```\n\n2. **Run the application**\n   ```bash\n   python app.py\n   ```\n\n## Usage\n\n1. Start the application by running `python app.py`\n2. Wait for the AI model to load\n3. Ask questions about Thalapathy Vijay\n4. Type 'quit', 'exit', or 'bye' to end the session\n\n## Example Questions\n\n- \"Tell me about Thalapathy Vijay's movies\"\n- \"What is TVK?\"\n- \"When did Vijay become Chief Minister?\"\n- \"What are his popular films?\"\n- \"Tell me about his dancing skills\"\n\n## How It Works\n\n1. **Knowledge Base**: Contains curated information about Thalapathy Vijay\n2. **Semantic Search**: Uses sentence transformers to find relevant information\n3. **Similarity Matching**: Compares user questions with knowledge base using cosine similarity\n4. **Response Generation**: Returns the most relevant information with confidence scores\n\n## Knowledge Areas\n\n- Acting career and filmography\n- Political journey and TVK\n- Personal achievements\n- Dancing and singing abilities\n- Social impact and philanthropy\n\n## Technical Details\n\n- **Model**: all-MiniLM-L6-v2 (sentence transformer)\n- **Similarity**: Cosine similarity for semantic matching\n- **Confidence Threshold**: 0.3 minimum for relevant answers\n\n## Commands\n\n- `quit` / `exit` / `bye` / `q` - Exit the application\n- `Ctrl+C` - Force quit\n\n## Future Enhancements\n\n- Web interface\n- Voice input/output\n- Expanded knowledge base\n- Multi-language support\n- Integration with external APIs
